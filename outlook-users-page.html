Outlook Users Page

The Users page provides a high-level overview on the status as well as statistics for emails sent and received by internal users. You can search for specific users and view data about their activities.

Tip

You can refresh the page by clicking the refresh icon beside the instance name of the app.

The main section of the page contains a table showing username, emails sent,emails from internal senders, groups, licensed or not, MDM-managed devices, and personally-owned devices.

The Users page allows you to search for specific users with filters. To use the search filters, click + Add Filter to apply one or more of these filters:

  • User Name: Find specific users.

  • Groups: Find internal users by the user group.

  • Licensed: Find internal users based on whether they are licensed or not.

  • Personal Devices: Find internal users by the type of OS on their personal device.

  • Managed Devices: Find internal users by the type of OS on their MDM-managed device.

The filters used determine which users are displayed in a table on the page. To view complete details about a user, click on a row to open the User Details summary.

The Users page also allows you to export the contents of the table. Click Export to download a spreadsheet file.
