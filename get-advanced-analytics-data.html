Get Advanced Analytics Data

These endpoints return the results for the Advanced Analytics reports and widgets specific to the Personal, Group, or Netskope Library.

The token should be passed to every REST API call for the tenant.

REQUEST ENDPOINT

https://<tenant-name>.goskope.com/api/v1/all_reports

This endpoint returns information about Reports for the account and related information as follows:

{
  "report_id": "5",
  "report_name": "MallaTestDashboard",
  "folder_id": "88",
  "folder_name": "old",
  "folder_type": "shared"
}

The purpose of the 'all_reports' API is to get the Report Name, Report ID, and Folder ID for an account and use this information to pass to following APIs:

https://<tenant-name>.goskope.com/api/v1/widgets_in_reports

https://<tenant-name>.goskope.com/api/v1/report_widget_data

RESPONSE

REQUEST ENDPOINT

https://<tenant-name>.goskope.com/api/v1/widgets_in_report

This endpoint returns all the widgets present inside a report. It includes following:

  • Report information

  • Number of Widgets

  • Widget Information

Key

Value

Description

report_id

positive integer

The report_id is derived from the following API:

https://<tenant-name>.goskope.com/api/v1/all_reports

If report_id is passed, folder_id is not mandatory.

report_name

URL encoded string

The report_name is derived from the following API:

https://<tenant-name>.goskope.com/api/v1/all_reports

If report_name is passed, folder_id is mandatory.

folder_id

positive Integer

The folder_id for a report is derived from the following API:

https://<tenant-name>.goskope.com/api/v1/all_reports

Either the report_id or report_name must be present.

RESPONSE

Sample URLs:

(report_id only)

https://<tenant-name>.goskope.com/api/v1/widgets_in_report?token=0e367f1ffe3dde821823f510ac22&amp;report_id=4829

(report_name with folder_id)

https://<tenant-name>.goskope.com/api/v1/widgets_in_report?token=0e367f1ffe3dde821823f510ac22&amp;folder_id=311&amp;report_name=Web%20Exec%20Dashboard%20Personal

REQUEST ENDPOINT

https://<tenant-name>.goskope.com/api/v1/report_widget_data

This endpoint returns information for a widget present in a report in CSV and JSON format. The row limit in the returned data is 5000.

Key

Value

Description

report_id

positive integer

The report_id is derived from the following API:

https://<tenant-name>.goskope.com/api/v1/widgets_in_report

If report_id is passed, folder_id is not mandatory.

report_name

URL encoded string

The report_name is derived from the following API:

https://<tenant-name>.goskope.com/api/v1/widgets_in_report

If report_name is passed, folder_id is mandatory.

widget_id

positive integer

The widget_id is derived from the following API:

https://<tenant-name>.goskope.com/api/v1/widgets_in_report

widget_name

URL encoded string

The widget_name is derived from the following API:

https://<tenant-name>.goskope.com/api/v1/widgets_in_report

folder_id

positive integer

The folder_id for a report is derived from the following API:

https://<tenant-name>.goskope.com/api/v1/widgets_in_report

result_format

csv | json

result format (default: csv)

Either the report_id or report_name must be present.

Either the widget_id or widget_name must be present.

RESPONSE

Sample URLs:

(report_id only with widget_id)

https://<tenant-name>.goskope.com/api/v1/report_widget_data?token=0e367f1ffe3dde821823f510ac22&amp;report_id=4829&amp;widget_id=29472

(report_id only with widget_name)

https://<tenant-name>.goskope.com/api/v1/report_widget_data?token=0e367f1ffe3dde821823f510ac22&amp;report_id=4829&amp;widget_name=Alerts%20-%20%23Events_ChangePrevPeriod

(report_name with folder_id and widget_id)

https://<tenant-name>.goskope.com/api/v1/report_widget_data?token=0e367f1ffe3dde821823f510ac22&amp;folder_id=311&amp;report_name=Web%20Exec%20Dashboard%20Personal&amp;widget_id=29472

(report_name with folder_id and widget_name)

https://<tenant-name>.goskope.com/api/v1/widgets_in_report?token=0e367f1ffe3dde821823f510ac22&amp;folder_id=311&amp;report_name=Web%20Exec%20Dashboard%20Personal
