Configure Netskope OAuth Token for Azure SCIM Integration

  1. Log in to the Netskope UI.

  2. Go to Settings > Tools > Directory Tools > SCIM Integration (tab) and click ADD TOKEN.

  3. Enter a name (like Azure) and click SAVE.

  4. Click OK to continue.

  5. The OAuth is created. Click on the displayed Token and copy the SCIM Server URL (Base URL) string for later use in the Azure configuration.

  6. Copy the OAuth Token (API Token) for later use in the Azure configuration, and then click OK.
