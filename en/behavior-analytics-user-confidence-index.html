Behavior Analytics User Confidence Index

The Behavior Analytics User Confidence Index page displays the top risky users based on their User Confidence Index. Admins will access this page to view details for each user.

To access this page, click Incidents > Behavior Analytics > User Confidence Index tab. The User Confidence Index tab is the default tab displayed in the Behavior Analytics overview page.

The User Confidence Index tab shows:

Total Users: The total number of users. Click the number of users to view the users in the User Confidence section of the page.

Poor: Total number of users with a Poor User Confidence Index score. Click the number to view details in the User Confidence section of the page.

Average: Total number of users with an Average User Confidence Index score. Click the number to view details in the User Confidence section of the page.

You can filter the information that displays in the User Confidence section by clicking the dropdown. The selection you make (All, Poor, Average, or Good) displays the list of users accordingly.

Select a listed user (1) to view details of the user in the right hand section (2).

Click the 'By Most Recent' dropdown to filter the view for the specified user.

By Most Recent: Displays the current information the system has for by the current information the system has for the specified user.

By UCI Change: Displays the most recent user confidence changes for the specific user.

Click the pop-out icon at the end of the user details to open the Incidents > Behavior Analytics > Incidents tab. See the Behavior Analytics Incidents topic for tab details.
