Issue Description

122439

API Protection

SharePoint API Protection

Fixed an issue related to detection of Sharepoint sites. This will reduce the time taken to detect newly created Sharepoint sites.

124532

API Protection

Salesforce API Protection

Fixed an issue related to API Protection for Salesforce, where "suspended" users would not show up correctly in the product UI.

124913

API Protection

Box API Protection

Fixed a bug in the API Protection for Box, where files owned by a "Service Account" were not listed even though internal collaborators had access to these files.

120190

CASB Inline Protection

Removed Sync activity from Amazon S3 app connector

When a block policy for Amazon S3 sync activity is in place, it is not blocking the sync operation even though there is a Block event.

112112

Data Protection

Special characters from dictionary not detected properly

With this fix, we have added a flag to enable characterset encoding for plain text files.

130089

IaaS

AWS, Azure, and GCP instance behavior change

AWS, Azure, and GCP instance name will check any forward and backward slashes in the instance name field and restrict the instance creation.

113769

NG SWG / CASB

Microsoft OneDrive for Business instance IDs

Some users see two different instance IDs for Microsoft OneDrive for Business. With this fix, these are now combined.

130191

SSPM

Security Posture page visible for certain users

Admin users with roles that only include access to Reports and not Security Posture can no longer view the API-enabled Protection > Compliance > Security Posture page.

125616

Steering

Netskope client disconnects

The Netskope client disconnects the tunnel when users lock their computer screens.

128293

Steering

Apps are not loading properly

Netskope client activates its internal module call 'proxy manager' to manage proxy destined network if Netskope client detects the proxy is in network.

With recent changes, the Netskope client removes 'proxy module' under certain conditions which leads to the problem. With this release, this is now fixed.

125608

Steering

Netskope client service crash

With this release, the Netskope client no longer crashes at boot time if the explicit proxy is configured at Web UI.
