Import JumpCloud LDAP Server Certificate into Windows

The Netskope Directory Importer requires the JumpCloud LDAP certificate to be installed on the Windows User certificate store for authentication to JumpCloud LDAP Directory service.

  1. Log in with admin credentials to the Windows device where the Netskope Directory Importer will be installed. For this purpose, a local admin account is being used (e.g. localadmin).

  2. Launch mmc.exe and select Add/Remove-Snapin from the File dropdown.

  3. Select Certificates and click Add.

  4. Select My User Account and click Finish.

  5. Select OK.

  6. Go to Certificates – Current User, right-click Personal, and then select All Tasks > Import.

  7. Current User will be pre-selected. Click Next.

  8. Click Browse and select the JumpCloud LDAP Server Certificate (Select All Files in the Dropdown to view the certificate) and click Next.

  9. Select Place all certificates in the following store > Personal and click Next.

  10. Click Finish and click OK.

  11. The JumpCloud Certificate will be displayed under Personal > Certificates.

  12. You can close the console at this time. Click No when prompted with the option to save console settings.
