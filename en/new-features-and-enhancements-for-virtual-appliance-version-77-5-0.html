New Features and Enhancements for Virtual Appliance Version 77.5.0

View forward proxy server certificate expiry date using SNMP

You can now view the expiry date of the forward proxy server certificate using SNMP. The following command provides the certificate expiry date.

snmpwalk -v2c -c <rocommunity-string> <appliance-ip> .1.3.6.1.4.1.8072.1.3.2.3.1.2
