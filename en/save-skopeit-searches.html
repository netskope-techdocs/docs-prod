Save SkopeIT Searches

Search queries can be saved as a filter in order to use them again.

To save a SkopeIT search:

  1. Create the search query by entering the query in the search field. For example, enter app eq Microsoft, and then select a Microsoft app from the dropdown list. Click Search.

  2. Click Save Filter.

    The Save Filters dialog box opens.

  3. Enter the name for your saved search and click Save.

    To use a previously created filter, click Filters above the search field and select a filter from the dropdown.
