Netskope GRE Configuration

To add a new GRE configuration:

  1. Log in to the Netskope tenant using admin credentials.

  2. Go to Settings > Security Cloud Platform > Traffic Steering > GRE.

  3. Select New GRE Configuration

  4. Enter a Configuration Name and the Source Peer IP. The Source Peer value should be the Source IP address of the Firewall interface where the GRE tunnel will be established from. If the GRE tunnel traffic is originating from multiple sites with different Source IP addresses, create a configuration per Source IP. When finished, and click Save and View POPS.

    .

  5. A list of Netskope POPs will be displayed. Select at least two (2) POPs from the list, ideally those closest to your location. Copy the GRE Gateway IPs and Probe IP Addresses. In this procedure, MEL1 and SY4 will be used as examples. Select Cancel when finished.

  6. Click Save to create the first tunnel. Create more tunnels by repeating these steps.

The GRE configuration is now complete and ready for the GRE tunnels to be established from Palo Alto Networks.
